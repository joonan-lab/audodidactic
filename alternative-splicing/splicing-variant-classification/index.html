스플라이싱 변이 분류

유전체 변이가 스플라이싱에 어떤 영향을 미치는지, 위치와 기능적 증거에 따라 어떻게 분류하는지 이해합니다. 위치 기반 분류에서 ACMG 기준, SpliceAI 점수 해석까지.

스플라이싱 변이 위치 기반 분류
핵심 원칙: 변이 위치는 분류의 출발점이지 결론이 아닙니다. 같은 canonical ±1,2 위치라도 RNA 기능 실험, SpliceAI 점수, 해당 전사체의 NMD 감수성, 질환과의 연관성을 종합해야 최종 분류가 가능합니다.
SpliceAI 점수 해석

SpliceAI는 deep neural network (잔차 CNN, 10,000 nt 수용 영역)를 이용해 각 뉴클레오타이드 위치가 donor 또는 acceptor 자리로 사용될 확률을 예측합니다. 변이의 delta score는 참조 서열 대비 스플라이싱 변화량을 나타냅니다.

Pangolin — 다조직 확장 버전

Pangolin (Zeng et al. 2022)은 GTEx 53개 조직의 조직 특이적 스플라이싱 데이터로 훈련된 SpliceAI의 확장 모델입니다.

  • 조직별 스플라이싱 영향 예측 가능
  • 뇌, 심장, 근육 등 조직 특이적 isoform 고려
  • SpliceAI보다 조직 맥락 정보 풍부
  • 희귀 조직 특이적 변이 해석에 유용

사용 시나리오

질환과 해당 조직의 스플라이싱이 다를 때. 예: 뇌 특이적 isoform이 임상적 의의가 있는 신경 발달 질환 변이. Pangolin 뇌 조직 점수가 SpliceAI보다 더 관련성 높은 예측 제공 가능.

ACMG/ClinGen 스플라이싱 변이 해석 기준

Richards et al. 2015 ACMG 기준과 Abou Tayoun et al. 2018 (ClinGen Splicing Subcommittee) 가이드라인에 기반합니다. 스플라이싱 변이는 PVS1 기준이 핵심입니다.

PVS1 적용 플로우차트 (Abou Tayoun 2018 기반)

PVS1 강도 결정 순서:

  1. 변이가 canonical ±1,2 또는 스플라이싱 영향 높음(SpliceAI≥0.8) 확인
  2. 해당 유전자가 LOF 질환 메커니즘인가? (haploinsufficiency)
  3. 영향받는 전사체가 NMD 감수성인가? (PTC 위치 확인)
  4. 해당 전사체가 임상적으로 관련 있는 전사체인가?
  5. 이전 exon의 마지막 코딩 영역이 중요한가?

스플라이싱 변이의 RNA 결과 유형

스플라이싱 변이 하나가 여러 RNA 결과를 동시에 만들 수 있습니다. RT-PCR 또는 RNA-seq으로 실제 결과를 확인하는 것이 중요합니다.

Exon Skipping

해당 exon 전체가 mRNA에서 제외. Out-of-frame skipping → NMD, in-frame skipping → 단백질 기능 변화. 가장 흔한 스플라이싱 변이 결과.

Intron Retention

인트론이 mRNA에 남음. 보통 PTC 도입 → NMD. 약한 스플라이스 부위 변이 또는 ESE 파괴 시 발생. 일부 경우 IR이 기능적 단백질 변이체 생성.

Cryptic Splice Site 활성화

기존에 사용되지 않던 스플라이스 부위가 활성화. 딥 인트로닉 변이의 주요 메커니즘. 새로운 exon(pseudoexon) 삽입 또는 exon 연장/단축.

Cryptic Exon (Pseudoexon)

딥 인트로닉 변이가 새로운 exon을 만들거나 기존 억제된 exon을 활성화. 보통 out-of-frame → NMD. ATM, BRCA2, DMD에서 다수 보고.

Exon Extension / Truncation

Alt 5'SS 또는 alt 3'SS 변이로 인해 엑손의 한쪽 경계가 변화. In-frame이면 단백질에 아미노산 추가/제거. Out-of-frame이면 NMD 가능.

Multiple Isoforms

하나의 변이가 여러 스플라이싱 결과를 동시에 만드는 경우. 일부 전사체에서 skipping, 다른 전사체에서 retention. 유전자별 전사체 분석 필요.

스플라이싱 변이 해석 워크플로우
STEP 1 — 위치 확인
  • Canonical ±1,2 → PVS1 가능성 바로 평가
  • Near-splice ±3–8 → 컨센서스 서열 점수 계산 (MaxEntScan)
  • Deep intronic → SpliceAI, MMSplice로 스크린
  • Exonic → SpliceAI ESE/ESS 영향, 동의어 변이인지 확인
STEP 2 — 계산 도구 적용
  • SpliceAI delta score 계산 (최우선)
  • MaxEntScan 스플라이스 부위 점수 (5'SS: MES5, 3'SS: MES3)
  • Pangolin 조직 특이적 점수 (해당 시)
  • MMSplice modular splice score
  • CADD-Splice 앙상블 점수
STEP 3 — RNA 실험 (필요 시)
  • RT-PCR 환자/대조군 cDNA → 이상 밴드 확인
  • Sanger 서열분석 이상 밴드 동정
  • RNA-seq 전장 스플라이싱 프로파일
  • Minigene assay 세포주에서 스플라이싱 재현
  • 환자 조직 RNA (혈액 → NMD 억제 후 분석)
STEP 4 — 전사체 분석
  • 영향받는 전사체가 임상적으로 관련 있는가? (MANE Select 등)
  • 해당 전사체가 NMD 감수성인가?
    (PTC가 마지막 exon junction보다 >50-55 nt 상류)
  • Skip/retention된 영역이 단백질 기능 도메인인가?
  • Rescue isoform이 존재하는가?
STEP 5 — 최종 분류 통합
