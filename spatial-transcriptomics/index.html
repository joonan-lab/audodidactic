L-R 분석 도구 비교:
• CellChat — 가장 널리 사용. 분비형/접촉형/ECM 경로 분류. Wang, FineST에서 사용.
• FineST — 조직학 이미지 + ST 융합, 핵 수준 해상도에서 L-R 분석. contrastive learning 기반.
• SPIDER — 공간적으로 가변적인 L-R 상호작용(SVI) 동정. 하류 전사인자 활성으로 기능적 검증.
• stLearn — Visium 공간 좌표 기반 L-R 통신 강도 추정. Wang에서 공간 검증에 사용.
• SpatialDM / CellPhoneDB / NicheNet — 추가 L-R/CCC 분석 도구.
