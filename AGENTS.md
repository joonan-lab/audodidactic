# Autodidact 작업 규칙 (Claude / Codex / 기타 에이전트 공용)

이 저장소는 **공개 GitHub Pages 사이트** (`https://joonan-lab.github.io/autodidact/`) 입니다. 외부에서 누구나 접근 가능하므로, 콘텐츠 추가 시 아래 규칙을 반드시 지켜 주세요.

## 1. 미출판 lab paper는 공개 페이지에 추가하지 않는다

**금지 대상**: `joonan-lab` 또는 `Joon-Yong An lab`이 저자로 참여하는 논문 중

- `in-preparation` (작성 중)
- `under-review` (저널 심사 중)
- `submitted` (투고 후 결정 대기)
- 공개 동의가 없는 `preprint`

이 상태의 논문은 본 사이트(또는 다른 공개 채널)에 절대 자동으로 추가하지 않습니다. 학술 윤리(저널 정책 위반 가능성)와 저자 동의 없는 결과 노출을 방지하기 위함입니다.

**출처 확인**: `~/Dropbox/References/llm-wiki/sources/*.md`의 frontmatter에서 `status` / `preprint` / `journal` / `source_collection: our-manuscript` 필드를 먼저 확인합니다. `our-manuscript` + 미출판 상태 조합은 곧바로 후보에서 제외합니다.

**용어 해석**: 사용자가 "출간 전 논문이 있는지 체크해 주세요"라고 하면, 이는 **이미 출판되었으나 페이지에 누락된 논문이 있는지** 확인하라는 뜻입니다. 미출판 자료를 올리라는 뜻이 아닙니다.

**예외**: 사용자가 명시적으로 "이 in-prep 논문은 페이지에 올려도 됩니다"라고 직접 허락한 경우에만 추가합니다 (자동 판단 금지).

## 2. 페이지 카운트는 변할 숫자를 명시하지 않는다

페이지 자체 컬렉션의 크기를 수치로 표시하지 않습니다 (예: "30+ 모델", "35편 논문", "16개 모델 비교", "9 sections"). 콘텐츠가 자라면서 변하기 때문에 추적·관리 비용이 큽니다. 외부 논문이 보고한 정량값(예: scPRINT-2의 350M cells, ASD risk gene 102개)은 사실이므로 그대로 보존합니다.

## 3. 자폐를 질병처럼 다루는 표현 회피

- "환자" → "당사자"
- "신경발달질환" → "NDD" (또는 "신경발달 형질")
- "동반질환" → "함께 나타나는 형질"
- "자폐 스펙트럼 장애 (ASD)" 공식 진단 명칭은 보존

## 4. 트랙 번호 같은 임시 순서는 박지 않는다

`Track 01`, `track-num` 같은 일련번호는 사용하지 않습니다. 콘텐츠가 추가되면서 순서가 바�뀌므로, 라벨로 식별합니다.

## 5. 한국어 본문 윤문은 보수적으로

`humanize-korean` skill 또는 유사 윤문을 적용할 때 변경률 5–15%를 넘지 않습니다. 도메인 술어(de novo, simplex, polygenic, exome 등), 수치, 인용(저자·연도·DOI), 직접 인용은 절대 건드리지 않습니다.
