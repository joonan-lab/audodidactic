통계 유전학

GWAS · Fine-mapping · Burden

Common SNP에서 rare variant까지, 그리고 GWAS peak를 causal variant로 좁히는 fine-mapping까지. 통계 모델이 어디서 신호를 잡고 어디서 막히는지를 조작해보며 익힙니다.

methodology · 방법론

유전 연관분석 방법 — GWAS, TADA, Burden, SKAT

Common SNP, de novo variant, rare coding burden, SKAT/SKAT-O, GLMM/REGENIE, LDSC/S-LDSC, MAGMA, PRS가 각각 어떤 신호를 포착하는지 설명하는 한글 인터랙티브. 대립유전자 빈도 / 효과 크기 지형도, 방법 선택기, GWAS·희귀변이·TADA 직관 콘솔, 분석 워크플로우, 핵심·예시 출처 링크를 포함합니다.

track 02

단세포 생물학

Atlas · Foundation Model

단일세포 시퀀싱이 인간 뇌를 어떻게 그려내는지, 그리고 그 위에 학습되는 foundation model이 어떤 어휘로 셀을 다루는지. atlas의 발견과 모델의 토큰화를 함께 봅니다.

atlas · timeline

인간 뇌 단일세포 시퀀싱 연구 지도

BrainSpan부터 BICCN, MERFISH, multiome, aging, Tri-IPC, cortical regionality까지 인간 뇌 single-cell/single-nucleus sequencing 연구를 연대별 발견과 셀타입 의미로 정리한 한글 학습 지도입니다. 그래픽 요약, 논문 타임라인, 하위 셀타입 quick map, regional-specific 발견 매트릭스, 검색 가능한 카탈로그를 포함합니다.

foundation model

단세포 Foundation Model — 55편의 풍경

scBERT(2022)부터 Tahoe-x1·scPRINT-2·Geneformer V2(2025-26)까지 단세포 RNA-seq foundation model의 4년 가속을 한 페이지로. 기초 4가지(scRNA-seq·transformer·토큰화·foundation model) → 8가지 토큰화 전략 풍경 → 16개 모델 비교 테이블(파라미터·corpus·context·objective·license) → HEIMDALL 토크나이저 분해(FG×FE×ORDER) → Reality Check 4편(Kedzierska·Csendes·Wei·Bereket) → Virtual Cell 1·2·3세대 비전.
